Проширени облик наредбе ``if else``:
====================================

Подсетимо се синтаксе наредбе ``if``:

.. activecode:: naredba_if_else
   :passivecode: true
   
   if uslove:
      naredba1()
      naredba2()
      ...
   else:
      naredbak1()
      naredbak2()
      ...
